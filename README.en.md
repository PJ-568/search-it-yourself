# Search It Yourself

> English | [简体中文](README.md)

Created for those who always ask simple questions that can be easily answered by searching or asking LLM AI.

A tool to quickly generate search instruction pages for humans who are unwilling or unable to search for themselves.

Compared to a complaint, "Search It Yourself" is more like a vow spoken to oneself, reminding oneself to persist on the path toward independence.

## Usage

1. Visit `index.html`;
2. (Optional) Enter custom search engine, format as `https://example.com/search?q=QUERY`;
3. Enter search content;
4. Click the search button;
5. Copy the generated link;
6. Send the link to humans who are unwilling or unable to search themselves.
