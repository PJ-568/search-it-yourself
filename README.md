# 自己搜去

> [English](README.en.md) | 简体中文

总有人问一些简单搜索或问大语言模型 AI 就能找到答案的简单问题。特做此项目。

给不愿意或不会自己搜索的人类，快速生成教他们搜索页面的工具。

比起一段抱怨，“自己搜去”更像是一段说给自己的誓言，提醒自己坚持走向独立的道路。

## 使用方法

1. 访问 `index.html`；
2. （可选）输入自定义搜索引擎，格式为 `https://example.com/search?q=QUERY` ；
3. 输入搜索内容；
4. 点击搜索按钮；
5. 复制生成的链接；
6. 转发给不愿意或不会自己搜索的人类。
